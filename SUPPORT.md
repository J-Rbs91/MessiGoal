# Obtenir de l'aide

Besoin d'aide avec MessiGoal ? Voici par où commencer.

- 📘 **Documentation** : [README](README.md), guide de contribution
  ([CONTRIBUTING](CONTRIBUTING.md)) et format des données
  ([data/README](data/README.md)).
- 💬 **Question ou idée** : ouvrez une
  [discussion](https://github.com/J-Rbs91/MessiGoal/discussions) (si activées) ou
  une [issue](https://github.com/J-Rbs91/MessiGoal/issues/new/choose).
- 🐛 **Bug** : utilisez le modèle d'issue « Signaler un bug ».
- ⚽ **Ajouter / corriger un but** : utilisez les modèles dédiés ou le bouton
  du site.
- 🔒 **Sécurité** : voir [SECURITY.md](SECURITY.md) (ne pas ouvrir d'issue
  publique).

Merci de garder les échanges courtois — voir le
[Code de conduite](CODE_OF_CONDUCT.md).
